== Description ==

This plugin generates Ipsum Text
based on the English translation
of Don Quixote de La Mancha
by Cervantes, as found in the
public domain on Project Gutenberg.

– Shortcode for character space
– Shortcode for number of words
– Shortcode for number of sentences
– Blogpost Generator

== Installation ==

1. Upload [`quixotix`](`https://github.com/JustinLawrenceMS/quixotix/archive/refs/tags/v1.0.0.zip`) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
